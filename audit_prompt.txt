# Claude Sonnet 4.6 Comprehensive Audit Prompt for Vantage Gaming (Katika)

---

## CONTEXT

You are auditing a production-grade P2P real-money skill gaming platform called Vantage Gaming (also known as Katika) for the Cameroonian market. The platform supports Chess, Checkers, Dice, Ludo, TicTacToe, Pool, and Cards (Whot) with real-money stakes via mobile money (MTN/Orange).

**Tech Stack:**
- Frontend: React 18 + TypeScript (strict) + Vite + Tailwind CSS + Framer Motion + PWA
- Backend: Node.js + Express + Socket.IO (monolithic server.js ~3300 lines)
- Database: Firebase Firestore (NoSQL)
- Auth: Firebase Auth (email/password, Google, anonymous)
- Payments: Fapshi API (MTN Mobile Money, Orange Money - XAF/FCFA)
- Testing: Vitest (10+ test suites)

---
